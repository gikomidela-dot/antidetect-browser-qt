SECTA Anti Detect - Linux x64
==============================

Installation:
1. Extract this archive
2. Install Qt6 dependencies:
   sudo pacman -S qt6-base qt6-webengine

3. Run:
   ./SECTAAntiDetect

Features:
- Multi-tab browser
- Fingerprint spoofing
- Proxy support
- Mass proxy import
- Cookie export/import
- Auto-save cookies

GitHub: https://github.com/gikomidela-dot/antidetect-browser-qt
